# World Population Exploratory Data Analysis (EDA)

## Overview
This project explores a global population dataset using Python and pandas. The analysis focuses on population growth trends, continent-level comparisons, correlations between variables, and visual data exploration through charts and heatmaps.

## Tools & Libraries
- Python
- pandas
- matplotlib
- seaborn
- Jupyter Notebook

## Key Tasks Performed
- Data inspection and cleaning
- Descriptive statistics analysis
- Correlation analysis using heatmaps
- Grouping and aggregation by continent
- Population trend analysis across decades
- Data visualization using line charts, boxplots, and heatmaps

## Key Insights
- Asia has the highest average population among continents.
- Population values across different years show very strong positive correlations.
- Growth rate and population density have relatively weak correlations with total population.
- Population growth trends become clearer when visualized across decades.

## Visualizations Included
- Correlation heatmap
- Population trend line charts
- Boxplots
- Grouped continent analysis
